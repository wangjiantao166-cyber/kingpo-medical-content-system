# wordpress-theme-template

## Purpose

Own block theme, templates, Gutenberg patterns, responsive behavior, technical tables, standard notices, gallery, downloads, and CTA display.

## Use Conditions

Use for front-end templates, design system, blocks, patterns, theme.json, and visual QA.

## Do Not Use Conditions

Do not change product facts, claim approvals, asset evidence status, or content model registration.

## Accepted Input Schema

- design system
- CPT/taxonomy spec
- asset manifest
- site publication records

## Output Schema

Template spec, block spec, theme scaffold, and front-end QA checklist.

## Blocking Conditions

Block if layout allows unsupported badges, hides reviewer notices, or breaks mobile usability.

## Handoff Rules

Send asset issues to `asset-visual-production`; send schema/index display issues to `technical-seo-link-graph`.
