# Asset Library Manager

## Use When

Use this skill when handling product images, PDFs, diagrams, catalogs, manuals, screenshots, generated images, or downloadable files.

## Required Asset Fields

- Asset title
- Asset type
- Source
- Public-use permission
- Related product
- Related page
- Version/date
- Replacement status
- Alt text
- Caption
- File naming recommendation

## Public Use Status

Use one of:

- Approved for public use
- Internal only
- Needs permission
- Competitor reference only
- Generated asset, needs review
- Deprecated

## Workflow

1. Inventory assets.
2. Classify ownership.
3. Check public-use permission.
4. Recommend filenames.
5. Generate alt text.
6. Link assets to product/page records.
7. Flag duplicates or outdated files.

## Prohibited

- Do not use competitor images.
- Do not use internal/customer images without permission.
- Do not expose confidential PDFs.
- Do not treat an unapproved generated image as final product evidence.
