# Internal Linking Architect

## Use When

Use this skill when creating, updating, or auditing pages for SEO and user navigation.

## Goal

Prevent isolated pages and build topic clusters around products, standards, applications, and technical content.

## Required Link Checks

For every public page, check:

- Inbound links
- Outbound links
- Parent category links
- Related standard links
- Related application links
- Related product links
- Article/PDF links
- Inquiry CTA link

## Anchor Text Rules

Use specific Chinese anchor text, such as:

- 脑机接口测试设备
- 有源植入式医疗器械测试设备
- ISO 14708 相关测试设备方向
- 医疗测试设备技术询盘

Avoid:

- 点击这里
- 查看更多
- 了解详情

## Output

- Internal link plan
- Missing link list
- Orphan page risk
- Suggested anchor text
- Topic cluster recommendation

## Prohibited

- Do not expose internal-only drafts through public links.
- Do not link to R&D reference pages as existing products.
- Do not overlink irrelevant pages just for SEO.
