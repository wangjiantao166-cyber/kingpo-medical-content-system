# AI Product Page Generator

## Use When

Use this skill when generating or improving an English product page draft from original product materials.

## Required Inputs

- Original product page or source material
- Product status if known
- Product category
- Known standards or test method direction
- Available technical specifications
- Available images/PDFs
- Any user-confirmed facts

## Generation Rules

Do not rigidly apply the same page template to every product.

Choose modules based on:

- Product type
- Test purpose
- Customer concerns
- Available source depth
- Overseas B2B buyer reading habits

Possible modules include:

- Product Overview
- Key Features
- Test Principle
- Standard Compliance / Standard Direction
- Applications
- Configuration Notes
- Selection Guide
- Technical Specifications
- FAQ

## Fact Control

All technical parameters must have a source.

If the source is incomplete:

- Use public information only for background context.
- Do not turn competitor data into KingPo facts.
- Mark uncertain fields as needing confirmation.
- Omit risky unsupported details.

## Writing Style

Write professional, clear, natural English for overseas B2B buyers.

Help customers quickly understand:

- What the equipment is
- What standard or test need it relates to
- What problem it solves
- What value it provides for procurement, labs, manufacturers, or certification bodies
- What configuration details need confirmation

Avoid keyword stuffing, empty marketing, overly academic language, unsupported compliance claims, and repetition.

## Output

Generate:

- H1
- SEO title
- Meta description
- URL slug
- Product overview
- Flexible product-specific sections
- Technical specifications table if sourced
- Standard/compliance wording
- Applications
- Selection/configuration notes
- FAQ
- Inquiry CTA
- Source notes
- Fields requiring human confirmation

## Prohibited

- Do not invent parameters.
- Do not invent standard versions or clauses.
- Do not invent certification, customer cases, prices, stock, delivery time, warranty, or regulatory/clinical claims.
- Do not copy competitor content.
