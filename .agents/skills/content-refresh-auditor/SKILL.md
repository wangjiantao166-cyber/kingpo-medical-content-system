# Content Refresh Auditor

## Use When

Use this skill for monthly, quarterly, or pre-launch audits.

## Audit Targets

- Product pages
- Category pages
- Standard pages
- Application pages
- Technical articles
- PDF downloads
- Images
- Inquiry pages

## Checks

- Last review date
- Broken links
- Missing inbound links
- Missing CTA
- Outdated PDF versions
- Old image assets
- Standards version risk
- Thin content
- Duplicate SEO title/meta description
- Product status mismatch
- Unverified claims

## Output

- Refresh priority
- Page/item list
- Issue type
- Recommended action
- Owner/reviewer
- Due date suggestion

## Prohibited

- Do not silently delete old content.
- Do not update technical facts without source evidence.
- Do not change URL structures without a URL plan.
