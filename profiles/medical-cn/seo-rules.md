# 医疗测试设备中文站 SEO 规则

## SEO 目标

本中文站的 SEO 目标是让搜索引擎和中文采购、研发、注册检验、第三方检测用户清楚理解 KingPo 在脑机接口、有源植入式医疗器械、医疗电气安全和医疗检测仪器测试设备方向的产品与技术能力。

重点不是堆关键词，而是建立：

- 产品与 GB / YY / ISO / IEC 标准的对应关系
- 测试设备与测试对象的对应关系
- 测试项目与研发验证、注册检验、第三方检测场景的对应关系
- 产品页、标准页、技术文章、专题页之间的内链关系
- 面向中文医疗器械客户的专业内容体系

## 第一阶段 SEO 方向

第一阶段中文 SEO 聚焦：

- 脑机接口测试设备
- 有源植入式医疗器械测试设备
- 医疗电气安全测试设备
- 神经刺激器测试设备
- 植入式医疗器械测试设备
- YY/T 1996 测试设备
- GB 16174 测试设备
- YY 0989 测试设备
- ISO 14708 测试设备
- EEG / EMG / EOG / LFP / Spike / ECoG 信号模拟
- 除颤兼容测试
- TMS 兼容性测试
- 超声兼容测试
- EMC 抗扰度测试

IP 防水、燃烧、电池、家电安规、普通量规等综合方向不作为第一阶段 SEO 重点。

## 页面 SEO 基础字段

每个重点页面必须准备：

- H1
- SEO Title
- Meta Description
- URL Slug
- 核心关键词
- 相关关键词
- 图片文件名建议
- 图片 Alt Text
- 内部链接建议
- FAQ
- Schema 建议
- 数据来源和人工确认状态

## SEO Title 规则

SEO Title 应包含：

- 产品核心名称
- 标准号或测试用途
- KingPo 或厂家属性

示例：

- 脑机接口测试设备｜KingPo 医疗检测设备
- 有源植入式医疗器械测试设备｜ISO 14708 测试方向
- YY/T 1996 测试设备｜有源植入式器械测试方案
- 电生理动态模拟闭环测试仪｜BCI 测试设备
- 植入式医疗器械除颤兼容测试仪｜医疗电气安全测试

不得写未经确认的“认证通过”“完全符合”“替代某品牌”等表述。

## Meta Description 规则

Meta Description 应简洁说明：

- 产品用途
- 相关标准或条款方向
- 测试对象
- 研发验证 / 注册检验 / 第三方检测价值
- 需要询盘确认的内容

示例：

KingPo 脑机接口测试设备用于电生理信号模拟、闭环刺激响应和测试数据记录，可服务 BCI 研发验证、注册检验准备和第三方检测场景。具体参数和标准条款需根据资料确认。

## URL Slug 规则

URL 建议使用英文、标准号和核心产品词。

示例：

- bci-test-equipment
- active-implantable-medical-device-test-equipment
- yy-t-1996-test-equipment
- gb-16174-test-equipment
- iso-14708-test-equipment
- electrophysiology-closed-loop-simulator
- eeg-dynamic-closed-loop-simulator
- implant-defibrillation-compatibility-tester
- tms-compatibility-test-system

不要使用中文 URL，避免编码混乱。

## H1 规则

H1 应自然、清晰，不要过长。

优先格式：

- 产品名称
- 标准号 + 测试设备
- 产品名称 + 测试用途

示例：

- 脑机接口测试设备
- 有源植入式医疗器械测试设备
- YY/T 1996 测试设备
- ISO 14708 有源植入式医疗器械测试设备
- 植入器械 TMS 兼容性测试仪

## 关键词规则

关键词应围绕真实搜索意图，不要机械重复。

关键词类型包括：

- 标准号关键词
- 产品名称关键词
- 测试项目关键词
- 测试对象关键词
- 研发验证关键词
- 注册检验关键词
- 第三方检测关键词
- 厂家 / 供应商关键词

示例：

- 脑机接口测试设备
- 有源植入式医疗器械测试设备
- 神经刺激器测试设备
- 植入式医疗器械测试设备
- YY/T 1996 测试设备
- GB 16174 测试设备
- ISO 14708 测试设备
- EEG 信号模拟器
- ECoG 信号模拟
- 除颤兼容测试设备
- TMS 兼容性测试仪
- 超声兼容测试仪
- EMC 抗扰度测试设备

## 内容质量规则

每个产品页必须提供真实价值，包括：

- 测试原理
- 适用标准与条款方向
- 测试对象和样品类型
- 测试项目
- 技术参数
- 研发验证 / 注册检验 / 第三方检测场景
- 选型建议
- FAQ
- 询盘 CTA

禁止：

- 空泛营销文案
- 关键词堆砌
- 复制英文站内容后简单翻译
- 复制同行内容
- 编造参数
- 编造标准版本
- 编造客户案例
- 编造认证资质或注册结论
- 编造库存、价格、交期
- 把研发参考产品写成 KingPo 已有现货产品

## 内链规则

每个产品页应尽量链接到：

- 相关产品
- 所属产品分类
- 相关标准页面
- 相关测试场景页面
- 相关技术文章
- 询盘页面

内链锚文本要自然。

示例：

- 查看更多有源植入式医疗器械测试设备
- 了解 YY/T 1996 测试设备方向
- 阅读脑机接口闭环刺激测试方案
- 查看电极耐压与组织接口测试相关设备

## 图片 SEO 规则

图片文件名建议使用英文、标准号、产品词和应用词。

示例：

- bci-test-equipment-laboratory.jpg
- active-implantable-medical-device-test-system.jpg
- yy-t-1996-test-equipment.jpg
- eeg-dynamic-closed-loop-simulator.jpg
- implant-defibrillation-compatibility-tester.jpg

Alt Text 应自然描述图片内容，不要堆关键词。

示例：

KingPo 脑机接口测试设备用于电生理信号模拟和闭环刺激测试

## FAQ 规则

每个重点产品页建议 5-8 个 FAQ。

FAQ 应回答真实客户问题，例如：

- 这台设备适用于哪些标准或条款方向？
- 适合测试哪些有源植入式器械或部件？
- 是否支持 EEG / EMG / EOG / LFP / Spike / ECoG 信号模拟？
- 是否可用于注册检验前研发验证？
- 是否支持定制测试方案？
- 询价时需要提供哪些参数和样品信息？

FAQ 不要写成广告语。

## 百度 SEO 注意事项

如果网站面向中国市场，应注意：

- 不依赖 Google Fonts
- 不依赖 Google reCAPTCHA
- 不嵌入 YouTube
- 图片和脚本尽量本地化
- 页面加载速度要稳定
- sitemap 正常生成
- robots.txt 不阻止重要页面
- 页面标题要符合中文搜索习惯
- 重要页面要有清晰内链

## 重复内容规则

中文站不能只是英文站、旧站或同行站点的翻译复制。

允许保留真实技术参数，但必须重新组织：

- 中文医疗器械研发和检测视角
- 测试原理解释
- 标准应用说明
- 研发验证和注册检验场景
- 选型建议
- FAQ
- 内链结构

如页面与现有站点重复风险高，应标记：

重复风险：高 / 中 / 低

并说明优化建议。
