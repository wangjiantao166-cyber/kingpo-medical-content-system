# 视觉资产与图片体系规划

## 当前阶段边界

本文件用于规划中文站视觉资产、图片类型、图片来源、合规边界和第一批图片准备清单。

当前阶段不连接服务器，不连接 WordPress，不修改现有网站，不发布页面，不生成完整产品页面。

## 图片来源分级

### P0: KingPo 公开官网 / PDF / 图片 / 内部确认资料

优先使用 KingPo / DGKingPo 已有公开官网、PDF catalog、产品图片、标准页面、技术文章和获批准内部资料。

P0 资料可用于证明 KingPo 已有医疗测试设备、IEC/ISO/YY/GB/UL 标准测试设备经验、产品页结构、PDF 风格、图片风格和 B2B 询盘路径。

P0 资料不能直接证明第一阶段 20 个新方向已经是成熟现货产品，除非老板或正式资料明确确认。

### P1: 官方标准 / 标准映射资料

用于标准映射、测试对象关系图、标准体系图和知识页配图。不得复制标准正文、条款或受版权保护图表。

### P2: 用户后续补充资料

包括老板确认的样机图、系统图、夹具图、测试场景图、PDF、产品路线表和可公开素材清单。

### P3: 同行参考

仅用于学习信息架构、视觉层级、模块组织和询盘路径，不得复制同行图片、图纸、PDF、软件界面、布局、参数表或品牌表达。

## 图片体系

### 1. Hero Image

用途：首页、分类页、核心方向页首屏。

建议内容：医疗测试实验室、测试设备工作台、信号采集与测试系统、工程师调试场景。

要求：专业、真实、技术感；不出现患者治疗场景，不暗示临床疗效。

来源优先级：KingPo 可公开实验室/设备图 > 获批准系统图 > 原创生成的实验室概念图。

### 2. Product / System Concept Image

用途：R&D 方向页、产品分类页。

建议内容：测试系统外观概念、模块组合、信号采集/输出模块、测试夹具组合。

要求：必须标记为概念图或示意图，不能让用户误认为现货产品外观。

### 3. Test Setup Image

用途：测试对象、测试项目和选型说明。

建议内容：样品、夹具、模拟负载、信号源、测量设备、数据记录模块之间的连接关系。

要求：优先使用获批准 KingPo 图；未批准时使用原创示意图。

### 4. Block Diagram / Workflow Image

用途：解释 BCI、闭环刺激、经皮能量传递、除颤兼容、TMS/超声/EMC 等测试链路。

建议内容：输入信号、样品、刺激输出、测量反馈、控制软件、数据记录。

要求：必须原创，不复制同行框图、软件界面或布局。

### 5. Standard Mapping Illustration

用途：标准页、知识页、R&D 页标准模块。

建议内容：标准族、测试对象、测试项目和页面使用等级之间的关系。

要求：不写未经确认标准版本、条款号和符合性声明。

### 6. Laboratory Scene Image

用途：首页、关于页、技术能力页、询盘页。

建议内容：医疗电气测试实验室、设备调试、工程测试台、文件和测试记录场景。

要求：不使用临床治疗、手术、患者图像。

### 7. Placeholder Image for R&D Directions

用途：20 个方向还未获批准公开图片时的占位。

建议内容：抽象但工程化的系统框图、模块化测试平台、信号链路示意。

要求：必须清楚标注“示意图 / 研发方向占位图”，避免误导为已量产外观。

## 图片风格规则

- 使用清晰、明亮、可检查的设备或系统图，不使用模糊暗色科技背景。
- 避免过度未来感、科幻脑机接口、医疗治疗场景和患者图。
- 图像应服务于理解测试对象、测试链路、标准映射和询盘准备。
- 对研发方向，图片优先为概念图、系统图、流程图和可公开实验室图。
- 图片文件名使用英文关键词和标准/方向词，例如 `bci-test-equipment-system-diagram.jpg`。
- Alt text 自然描述图片内容，不堆关键词，不写未确认产品能力。

## 现有官网可学习的图片和模块风格

DGKingPo 现有医疗测试设备页面可参考：

- 产品主图 + 图库结构
- PDF 下载模块
- FAQ 模块
- 询盘按钮和联系入口
- 产品分类页卡片结构
- 标准号、测试对象和应用场景的呈现方式

可学习的是结构和资料组织方式，不等于复制图片或直接复用旧站页面文案。旧图是否可用于新中文站，需要确认版权、清晰度、是否最新和是否适合医疗专站定位。

## 禁止图片使用

- 不复制 Fluke、WhaleTeq 或其他同行图片。
- 不复制同行 PDF 封面、图纸、软件界面、参数表截图或页面布局。
- 不使用未授权客户现场图、客户名称、案例图。
- 不使用患者治疗、手术、临床效果暗示图。
- 不把概念图伪装成已量产设备照片。

## 交付建议

第一批先准备 8-12 张核心图片和 3 类可复用图表模板：标准映射图、测试链路图、研发方向占位图。等老板确认内部样机、夹具和系统图后，再替换占位图。
