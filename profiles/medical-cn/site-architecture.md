# 医疗专站信息架构规划

## 项目定位

本项目用于规划和建设 KingPo 全新的独立中文 B2B 医疗测试设备网站，重点聚焦脑机接口与有源植入式医疗器械测试设备。

项目名称：脑机接口与有源植入式医疗器械测试设备中文站

英文定位：BCI and Active Implantable Medical Device Test Equipment Website

该中文站是新站项目，不迁移旧站，不替换旧站，不创建 301 跳转，不修改任何现有公司网站。现有网站仅可作为只读资料来源或只读同行参考来源。

## 目标用户

网站内容面向中文医疗测试设备 B2B 用户，包括：

- 脑机接口研发团队
- 有源植入式医疗器械厂家
- 神经刺激器和 IPG 研发团队
- 起搏器 / ICD 相关研发和检测团队
- 医疗器械注册检验机构
- 第三方检测机构
- 医疗器械企业实验室
- 高校、医院和科研单位
- 质量、法规、可靠性和研发验证部门

## 第一阶段产品范围

第一阶段只聚焦：

- 脑机接口测试设备
- 有源植入式医疗器械测试设备
- 神经刺激器测试设备
- 植入式起搏器 / ICD 测试设备
- EEG / EMG / EOG / LFP / Spike / ECoG 信号模拟
- 闭环神经刺激测试
- 经皮能量传递测试
- 除颤兼容测试
- EMC / 磁场 / TMS / 超声兼容测试
- 电极、电导线、绝缘、耐压、组织接口测试
- GB 16174 / YY 0989 / YY/T 1996 / ISO 14708 相关测试设备

## 移出第一阶段的旧方向

以下旧方向不再作为第一阶段重点：

- IP 防水测试设备
- 燃烧测试设备
- 电池测试设备
- 家电安规测试设备
- 普通量规
- 插头插座量规
- IEC 61032 通用试验探棒
- 普通综合工业测试设备

医疗量规、ISO 80369、ISO 18250、鲁尔接头测试设备暂时不纳入第一阶段主线，后续可作为“医疗连接件与医疗量规专题”单独补充。

## 建站原则

- 优先采用 WordPress 作为中文站技术路线。
- 优先在独立测试环境完成结构、模板、内容和表单验证。
- 不直接修改生产网站。
- 不更改 DNS。
- 不发布正式页面。
- 不创建 301 跳转。
- 不删除、覆盖或替换旧站 URL。
- 不编造产品参数、标准版本、客户案例、认证资质、注册结论、临床结论、价格、库存或交期。
- 所有关键数据必须标记来源；无法确认的数据标记为“需要人工确认”。
- 研发参考方向必须标记“研发参考 / 需要人工确认”。

标准页面应帮助用户按标准查找设备。第一阶段重点围绕：

- GB 16174
- YY 0989
- YY/T 1996
- ISO 14708
- IEC 60601 相关方向

不得编造标准版本或条款解释。标准版本、条款方向和适用产品必须有来源或人工确认。

- FAQ

所有资料应基于真实来源或用户确认输入，不得生成无法核实的技术结论。

## URL 原则

当前阶段只规划新站 URL，不迁移旧站 URL，不建立旧站到新站的 301 跳转。

建议 URL 使用清晰英文短路径，例如：

- `/products/`
- `/products/bci-test-equipment/`
- `/products/active-implantable-medical-device-test-equipment/`
- `/standards/iso-14708/`
- `/standards/yy-t-1996/`
- `/applications/closed-loop-neurostimulation-testing/`
- `/technical-resources/`
- `/contact/`

所有 slug 在发布前需人工复核。

## 内容质量要求

内容必须准确、可追溯、医疗语境谨慎、采购和研发友好。禁止薄内容、关键词堆砌、夸大性能、虚构案例、虚构认证、虚构注册结论或虚构临床结论。
