# KingPo 中文站项目说明

## 项目名称

脑机接口与有源植入式医疗器械测试设备中文站

BCI and Active Implantable Medical Device Test Equipment Website

## 项目目标

搭建一个面向中文 B2B 医疗检测设备客户的专业网站，重点聚焦脑机接口、有源植入式医疗器械、神经刺激器、植入式起搏器 / ICD、医疗电气安全和医疗检测仪器相关测试设备。

网站应支持：

- 产品展示
- 中文产品询盘
- 标准与测试条款说明
- 研发验证和注册检验场景说明
- 第三方检测和实验室应用说明
- PDF 资料下载
- SEO 内容扩展
- 后续产品资料维护

## 网站定位

本网站不再定位为综合工业测试设备站，也不再以 IP 防水、燃烧、电池、家电安规、普通量规等方向作为第一阶段重点。

本网站定位为：

中文 B2B medical test equipment website focused on BCI and active implantable medical device test equipment.

内容应更符合中文医疗器械研发、注册检验、第三方检测和采购用户的阅读习惯，重点解释设备用于什么医疗测试场景、对应哪些 GB / YY / ISO / IEC / ANSI/AAMI / ASTM / IEEE 标准或条款方向、如何支持研发验证、设计确认、注册检验或第三方检测。

## 第一阶段核心产品方向

第一阶段重点产品方向包括：

- 脑机接口测试设备
- 有源植入式医疗器械测试设备
- 神经刺激器测试设备
- 植入式起搏器 / ICD 测试设备
- EEG / EMG / EOG / LFP / Spike / ECoG 信号模拟
- 闭环神经刺激测试
- 经皮能量传递测试
- 除颤兼容测试
- EMC / 磁场 / TMS / 超声兼容测试
- 电极、电导线、绝缘、耐压、组织接口测试
- GB 16174 / YY 0989 / YY/T 1996 / ISO 14708 相关测试设备

## 国际标准体系

除 GB 16174、YY 0989、YY/T 1996、ISO 14708 外，项目需要建立 IEC、ISO、ANSI/AAMI、ASTM、IEEE 等国际标准映射体系。

标准体系详见：`profiles/medical-cn/international-standards-landscape.md`。

每个产品方向必须先做标准映射，再决定是否可以写页面。标准映射字段必须包含：

- Standard Number
- Standard Name
- Organization
- Relevance to Product Direction
- Test Object
- Possible Test Item
- Page Use: 主标准 / 辅助标准 / 背景标准 / 不适用
- Evidence Source
- Human Verification Required

不得把标准号写成 KingPo 产品已完全符合，除非有明确资料支持。不得编造标准版本、条款号或测试条件。

## 降级或移出第一阶段的旧方向

以下旧方向不再作为第一阶段重点：

- IP 防水测试设备
- 燃烧测试设备
- 电池测试设备
- 家电安规测试设备
- 普通量规
- 插头插座量规
- IEC 61032 通用试验探棒
- 普通综合工业测试设备

医疗量规、ISO 80369、ISO 18250、鲁尔接头测试设备暂时不纳入第一阶段主线。后续可作为“医疗连接件与医疗量规专题”单独补充。

## 内容原则

所有内容必须专业、准确、清晰，不得编造技术参数、标准版本、条款号、客户案例、认证资质、注册结论、临床结论、价格、库存或交期。

产品参数必须来自新上传的表格/PDF 清单、已有官网、PDF 目录、产品图片、技术文件或人工确认资料。

资料不清楚、冲突或仅属于研发方向时，必须标记为“需要人工确认”或“研发参考 / 需要人工确认”。

不得把 Fluke、WhaleTeq 或其他同行资料当成 KingPo 产品事实来源。
