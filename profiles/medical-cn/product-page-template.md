# 医疗测试设备中文产品页模板规则

## 页面目标

中文产品页面向医疗器械研发、注册检验、第三方检测、企业实验室和科研用户，不是英文页面的简单翻译。

页面应帮助客户快速理解：

- 设备用于什么医疗测试场景
- 对应哪些 GB / YY / ISO / IEC 标准或条款方向
- 适用于哪些器械、部件、样品、信号或组织接口对象
- 主要测试项目是什么
- 测试原理如何理解
- 如何支持研发验证、设计确认、注册检验或第三方检测
- 技术参数是否有来源
- 哪些内容需要人工确认
- 如何发起技术询盘

## 适用项目范围

第一阶段产品页优先服务：

- 脑机接口测试设备
- 有源植入式医疗器械测试设备
- 神经刺激器测试设备
- 植入式起搏器 / ICD 测试设备
- EEG / EMG / EOG / LFP / Spike / ECoG 信号模拟
- 闭环神经刺激测试
- 经皮能量传递测试
- 除颤兼容测试
- EMC / 磁场 / TMS / 超声兼容测试
- 电极、电导线、绝缘、耐压、组织接口测试
- GB 16174 / YY 0989 / YY/T 1996 / ISO 14708 相关测试设备

IP 防水、燃烧、电池、家电安规、普通量规等综合方向不作为第一阶段产品页主线。

## 固定页面结构

所有中文产品页必须按以下顺序：

1. H1
2. 一句话产品定位
3. 核心信息
4. 产品概述
5. 适用标准与条款方向
6. 测试原理
7. 测试对象与样品类型
8. 测试项目
9. 技术参数
10. 典型应用
11. 研发验证 / 注册检验 / 第三方检测场景
12. 选型与配置建议
13. 相关产品
14. 常见问题
15. 询盘 CTA

“核心信息”必须放在“产品概述”之前。

## H1 规则

H1 应清晰表达产品名称和核心医疗测试用途。

示例：

- 脑机接口测试设备
- 有源植入式医疗器械测试设备
- 电生理动态模拟闭环测试仪
- EEG 动态模拟闭环测试仪
- 植入式医疗器械除颤兼容测试仪
- 心律管理器械低频电磁抗扰度测试仪

不要在 H1 中堆砌过多关键词，不要写未经确认的“已通过认证”“完全符合”等表述。

## 一句话产品定位

H1 后必须用 1-2 句话说明设备用途。

示例：

本设备用于脑机接口与有源植入式医疗器械研发验证中的电生理信号模拟、闭环刺激响应和测试数据记录，可用于研发实验室、注册检验准备和第三方检测场景。具体测试参数和适用标准需以产品资料或人工确认为准。

## 核心信息

核心信息建议包含：

- 适用标准或条款方向
- 测试对象
- 主要测试目的
- 典型使用场景
- 信号 / 电气 / 机械 / 兼容性测试方向
- 数据来源
- 需要人工确认项

## 产品概述

产品概述应说明设备的测试背景、使用目的和客户价值。

应重点解释：

- 为什么该测试对 BCI 或有源植入式医疗器械重要
- 它解决什么研发验证或检测问题
- 实验室如何使用
- 对注册检验、第三方检测或企业研发有什么价值

避免空泛营销语，例如“国际领先”“品质卓越”“深受客户好评”。

## 适用标准与条款方向

列出设备相关的标准或条款方向。

重点标准方向包括：

- GB 16174
- YY 0989
- YY/T 1996
- ISO 14708
- IEC 60601 相关方向
- 其他经资料或用户确认的医疗器械测试标准

要求：

- 标准号必须准确
- 不确定版本时不要写死版本号
- 不得编造标准条款
- 不得声称完全符合全部条款，除非有明确来源
- 如果资料不完整，标记“需要人工确认”

## 测试原理

用中文解释测试方法背后的原理，让采购、研发和检测用户能理解设备作用。

可根据产品类型解释：

- 电生理信号模拟与采集
- 闭环刺激输入输出响应
- 经皮能量传递过程模拟
- 除颤兼容或高能量脉冲影响
- 静磁场、时变磁场、TMS、超声或 EMC 抗扰度影响
- 电极、电导线、绝缘、耐压和组织接口网络模拟
- 机械应力、压力循环或头部植入环境模拟

不得编造设备不具备的原理或测试能力。

## 测试对象与样品类型

说明设备适用的对象，例如：

- 脑机接口系统或部件
- 有源植入式医疗器械
- 神经刺激器
- 植入式脉冲发生器 IPG
- 植入式起搏器 / ICD 相关器械
- 电极、电导线、连接部件和绝缘结构
- EEG / EMG / EOG / LFP / Spike / ECoG 信号链路
- 组织接口等效网络或模拟负载

如果样品对象不确定，标记“需要人工确认”。

## 测试项目

根据产品资料列出真实测试项目。

不得编造设备不具备的功能。如果测试项目来自新上传表格、PDF 或旧资料，应保留来源记录。

## 技术参数

技术参数必须结构化展示。

医疗测试设备常见字段包括：

- 信号类型
- 通道数
- 输出范围
- 频率范围
- 波形或脉冲参数
- 电压 / 电流范围
- 采样率
- 同步 / 触发方式
- 负载或等效网络
- 绝缘 / 耐压能力
- 机械应力或压力循环范围
- 磁场 / 电场 / 超声 / EMC 条件
- 软件功能
- 数据记录方式
- 校准或溯源要求
- 标准附件和可选附件

如果参数不明确，写“需要人工确认”。

## 典型应用

应用场景应贴近真实用户：

- 医疗器械研发验证
- 有源植入式器械设计确认
- 注册检验准备
- 第三方检测机构
- 企业实验室
- 高校、医院和科研单位
- 质量、法规和可靠性部门

## 研发验证 / 注册检验 / 第三方检测场景

说明设备如何帮助用户完成：

- 研发阶段功能验证
- 设计输入和设计输出验证
- 标准条款预评估
- 注册检验前摸底
- 第三方检测方案配置
- 实验室测试平台搭建

不得暗示产品已获得监管认可，除非有明确来源。

## 选型与配置建议

询盘时建议客户提供：

- 适用标准和条款方向
- 样品类型
- 信号类型和通道数
- 测试项目
- 输出 / 采集范围
- 是否需要闭环控制
- 是否需要同步触发或数据记录
- 是否涉及除颤、TMS、超声、磁场、EMC 等兼容性场景
- 是否需要校准或溯源文件
- 实验室安装条件

## 相关产品

推荐相关设备、模块、夹具、模拟器或测试系统。

相关产品必须真实存在或明确标记为“研发参考 / 需要人工确认”。不得虚构现货产品。

## 常见问题

每个产品页建议 5-8 个 FAQ，回答真实客户关心的问题：

- 该设备适用于哪些标准或条款方向？
- 适合测试哪些医疗器械或部件？
- 是否支持 EEG / EMG / EOG / LFP / Spike / ECoG 信号模拟？
- 是否支持闭环刺激测试？
- 是否可用于注册检验前研发验证？
- 询价时需要提供哪些样品和测试条件？
- 是否可以按测试方案定制？
- 哪些参数需要人工确认？

## 询盘 CTA

页面底部应有简洁询盘引导。

示例：

如需确认该设备是否适用于您的 BCI、有源植入式医疗器械或医疗电气安全测试项目，请联系 KingPo 技术团队。请提供适用标准、样品类型、测试项目、信号或电气参数范围，以及是否涉及注册检验、研发验证或第三方检测场景。
