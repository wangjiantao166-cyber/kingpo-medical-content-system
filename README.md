# KingPo Medical Content System

KingPo Chinese B2B medical test equipment website planning and product content system.

This repository is a working system for building a WordPress-based product knowledge website similar in information architecture depth to mature B2B equipment websites, while keeping KingPo content original, source-based, and safe for medical/regulatory claims.

The repository currently contains:

- Project rules and source ownership policy
- WordPress content system blueprint
- Product, standard, application, article, asset, and inquiry field models
- Internal linking and SEO architecture
- Agent/skill workflows for product onboarding, standards review, WordPress drafting, QA, asset management, refresh audits, and data import/export

This repository does not directly modify a production website. All implementation should start in a staging WordPress environment.
