# Asset Rights and Visual Production

## Asset Types

```text
real_product_photo
real_test_photo
technical_diagram
workflow_diagram
render
AI_concept_image
customer_asset
competitor_reference
certificate
manual
catalog
test_report
```

## Public Use Rules

- `competitor_reference` assets are never public.
- `customer_asset` requires explicit permission.
- `AI_concept_image` must be labeled as generated and manually reviewed.
- AI images cannot be used as evidence of real product structure, function, configuration, certification, installation, or customer use.
- Certificates, reports, and manuals must preserve version, source, owner, and permission status.
- Product main image, detail image, diagram, social image, and PDF cover need different size and crop specifications.

## Visual Production Responsibilities

The visual agent or designer should provide:

- image type;
- intended page placement;
- source or generation prompt;
- public-use status;
- alt text;
- caption;
- related entity IDs;
- reviewer;
- replacement or expiry note.
