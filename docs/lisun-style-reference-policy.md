# B2B Equipment Website Reference Policy

## Purpose

This project may study strong B2B equipment websites, including LISUN, for information architecture, navigation strategy, product catalog organization, internal linking, and conversion flow.

Reference does not mean copying.

## Allowed Reference Use

It is allowed to study:

- Navigation structure
- Product category depth
- Product page module order
- Application page concept
- Standard page concept
- Installation/case page concept
- Internal linking patterns
- Inquiry CTA placement
- PDF/download handling patterns
- SEO topic cluster strategy

## Prohibited Use

Do not copy:

- Text
- Images
- Diagrams
- Tables
- Product models
- Product parameters
- Customer names
- Case studies
- Certifications
- Page layout details
- HTML/CSS
- Screenshots
- Manuals or PDFs
- Brand expressions

Do not imply KingPo has the same product, same certification, same customer case, or same test capability unless verified by KingPo-owned or user-confirmed sources.

## What to Build Instead

Build a KingPo-specific structure:

- Medical test equipment focused categories
- Standards mapping for GB / YY / ISO / IEC / ANSI/AAMI / ASTM / IEEE
- BCI and active implantable medical device application pages
- Source-recorded product fields
- Cautious medical/regulatory wording
- Inquiry forms designed for test object, standard, sample, and parameter collection

## Safe Benchmarking Output

When an agent benchmarks another site, it should output:

- Observed information architecture pattern
- Why the pattern is useful
- KingPo-safe adaptation
- Copying risk
- Required original assets or data

It should not output copied page text or scraped content for reuse.
