# 产品来源优先级规划

## 目的

本文件定义 KingPo 医疗专站产品资料采集的来源优先级、标准映射优先级和第一批产品方向筛选规则。

当前阶段只做资料和标准映射规划，不连接服务器，不修改网站，不改 DNS，不发布页面，不创建 301，不创建网站代码，不生成最终产品页面。

## 新项目定位

脑机接口与有源植入式医疗器械测试设备中文站

BCI and Active Implantable Medical Device Test Equipment Website

第一阶段专攻医疗方向，尤其是脑机接口、有源植入式医疗器械、医疗电气安全和医疗检测仪器相关测试设备。

## 来源优先级

### P0：新上传表格 / PDF 清单 / 用户确认资料

第一阶段核心产品路线优先基于用户新上传的表格、PDF 清单、产品路线资料和人工确认信息。

P0 来源用于确认产品方向、产品名称、型号、标准号、标准版本、测试对象、测试项目、技术参数、是否已有产品、是否研发参考、是否可作为样板页。

### P1：标准映射证据

每个产品方向必须先建立标准映射，再决定是否进入页面草稿。

标准映射证据包括：

- 官方标准资料或用户提供的标准清单
- 新上传表格/PDF 中的标准字段
- KingPo 产品文件中的标准说明
- 人工确认的标准适用性

当现有资料不完整时，应优先参考官方或权威标准机构公开资料，例如 IEC、ISO、EN / CEN / CENELEC / ETSI、UL、UN ECE、SASO、BIS、ANSI/AAMI、ASTM、IEEE 等官方标准目录、标准说明、公开 scope 或官方简介。

这些资料只能用于确认标准编号、标准名称、组织、标准族、公开适用范围和高层级相关性，不能自动证明 KingPo 产品符合该标准、通过认证、覆盖全部条款或适用于注册检验。

标准映射字段必须包含：

- Standard Number
- Standard Name
- Organization
- Relevance to Product Direction
- Test Object
- Possible Test Item
- Page Use: 主标准 / 辅助标准 / 背景标准 / 不适用
- Evidence Source
- Human Verification Required

如果标准适用性不确定，标记为“需要人工确认”，不要写进最终产品页。

### P2：现有官网主要事实来源

以下网站是现有产品资料、产品分类、图片、PDF、询盘路径和公司能力说明的主要参考来源。使用时仍需记录具体 URL、页面标题、字段来源和人工确认状态。

1. https://www.dgkingpo.com/，KingPo / DGKingPo 主站和综合测试设备事实来源
2. https://www.batterytestingmachine.com/，KingPo 英文产品站和电池测试、环境试验箱、安全合规、医疗测试设备等现有资料来源
3. https://www.kingpocn.com/，KingPo 国内中文站资料来源，可用于中文产品事实、分类、图片、公司能力和中文表达参考；用于外贸英文站时必须重新改写为海外 B2B 语境，不能直接机翻照搬
4. https://www.kingpo.hk/，KingPo 品牌或公司相关资料来源，需结合页面内容确认用途
5. https://www.syrianetf.com/，仅作为旧资料参考源，不能作为生产目标

关键参数仍必须标记具体来源。

### P3：待确认内部线索

来源不完整或版本不明确的资料可作为线索，但不能直接生成确定内容。必须标记为“需要人工确认”。

### P4：同行参考来源

Fluke Biomedical、WhaleTeq 和其他同行资料只能用于市场学习、测试项目理解、产品路线参考、页面结构、FAQ 方向、用户需求和研发参考，不得作为 KingPo 产品事实来源或产品参数来源。

## 国际标准体系

标准体系详见：`docs/international-standards-landscape.md`。

第一阶段应考虑以下标准族：

- IEC 60601 基础与并列标准
- IEC 60601 EEG / ECG / monitoring / defibrillation / pacing / MRI / ultrasound 相关专用标准
- ISO 14708 有源植入式医疗器械系列
- ISO 10993 / ISO 14971 / ISO 13485 / ISO 14155 / ISO 15223-1 / ISO 20417 等通用医疗器械标准
- IEC 61000-4-x EMC / 磁场 / 抗扰度相关标准
- ANSI/AAMI EC12、EC13、DF80、PC69 等仅在适用且来源确认后使用
- ASTM F-series 仅在产品对象确认后使用
- IEEE / ISO/IEEE 11073 仅用于通信或数据接口相关页面

以下方向不再作为第一阶段重点：IP 防水测试设备、燃烧测试设备、电池测试设备、家电安规测试设备、普通量规、插头插座量规、IEC 61032 通用试验探棒、普通综合工业测试设备。

医疗量规、ISO 80369、ISO 18250、鲁尔接头测试设备暂时不纳入第一阶段主线。后续可作为“医疗连接件与医疗量规专题”单独补充。

## Source Record 字段

每个产品方向进入内容制作前，应建立 source record：

- 产品方向
- 产品中文名称
- 产品英文名称
- 型号
- 来源等级
- P0 表格 / PDF 来源
- 标准映射表
- 我方事实来源 URL
- 我方事实来源文件
- 旧资料参考 URL
- 同行参考 URL
- 可确认字段
- 需要人工确认字段
- 是否研发参考
- 是否可作为 KingPo 现货产品页面
- 数据冲突说明
- 是否可进入样板页制作

## 禁止规则

- 不得把同行参数作为 KingPo 参数。
- 不得把同行案例作为 KingPo 案例。
- 不得把同行认证声明作为 KingPo 认证。
- 不得复制同行图片、PDF、正文、型号体系、截图、图纸、软件界面或布局。
- 不得生成未经来源确认的技术参数。
- 不得编造标准版本、条款号或测试条件。
- 不得编造价格、库存、交期。
- 不得把研发参考产品写成 KingPo 已有现货产品。
