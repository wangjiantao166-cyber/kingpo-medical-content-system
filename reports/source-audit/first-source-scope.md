# 第一轮资料来源采集范围

## 当前阶段说明

本报告定义脑机接口与有源植入式医疗器械测试设备中文站的第一轮资料来源审计和采集范围。当前阶段仅制定采集范围和使用规则，尚未访问或采集这些网站内容。

限制：

- 不连接服务器。
- 不修改任何网站。
- 不登录任何网站后台。
- 不改 DNS。
- 不发布页面。
- 不创建 301。
- 不创建网站代码。
- 不绕过 robots.txt 或访问限制。
- 不高频请求网站。
- 不复制同行内容。
- 不把同行参数当成 KingPo 参数。
- 不生成未经来源确认的产品技术参数。
- 不把研发参考产品写成 KingPo 已有现货产品。

## 第一阶段核心来源

第一阶段最高优先级来源是用户新上传的表格、PDF 清单、产品路线资料和人工确认信息。

优先提取：

- 产品名称
- 型号
- 标准号
- 标准版本
- 标准条款方向
- 测试对象
- 样品类型
- 测试项目
- 技术参数
- 是否已有产品
- 是否研发参考
- 是否适合做样板页

## 需要人工确认的问题

1. 新上传表格/PDF 清单中每个产品是否为 KingPo 已有产品、研发中产品或研发参考方向。
2. GB 16174 / YY 0989 / YY/T 1996 / ISO 14708 的具体标准版本和适用条款。
3. 每个产品的型号、技术参数、信号范围、电气范围、机械/压力/磁场/超声/EMC 条件。
4. 是否可以公开产品图片、系统图、PDF 和技术资料。
5. 哪些页面可以作为第一批样板页。
6. 哪些产品只能标记为“研发参考 / 需要人工确认”。
7. 中文站品牌名称、公司名称和联系方式的标准写法。
8. 是否有指定的测试站域名或独立 WordPress 安装计划。
