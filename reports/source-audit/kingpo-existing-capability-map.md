# KingPo Existing Capability Map

## 当前阶段边界

本文件用于把 KingPo / DGKingPo 已有公开官网资料与第一阶段 20 个研发方向建立“背景能力映射”。

现有官网资料可以证明 KingPo 已有医疗测试设备、标准测试设备、产品页结构、PDF、图片和询盘路径基础，但不能直接证明第一阶段 20 个方向已经是成熟现货产品。

第一阶段 20 个方向仍默认按“研发中 / 技术预研 / 可定制开发评估 / 需要人工确认”处理。

## 可以先做研发方向页草稿的方向

在不写成现货、不写确认参数、不写完全符合标准的前提下，以下方向可以优先做内部草稿：

1. 脑机接口测试设备
2. 有源植入式医疗器械测试设备
3. EEG 动态模拟闭环测试仪
4. 电生理动态模拟闭环测试仪
5. 电极耐压自动测试仪
6. 神经刺激器电极组织接口网络模块

这些方向可以用 KingPo 现有医疗测试设备、ECG/电极、除颤、电气安全页面作为背景能力支撑，但页面必须保留“研发方向 / 参数待确认 / 老板待确认”。

## 必须等老板确认后再推进公开页面的方向

1. 植入式医疗器械除颤兼容测试仪
2. 心律管理器械低频电磁抗扰度测试仪
3. 植入器械 TMS 兼容性测试仪
4. 大功率电场模拟器
5. 静磁场测试系统
6. 时变磁场测试仪
7. 超声兼容测试仪
8. 超声脑机接口性能测试仪

这些方向涉及高能量、电磁、磁场、TMS、超声或心律管理器械安全边界，公开前必须确认技术资料、风险边界和标准适用性。

## 可从现有官网学习的模板和图片风格

- 医疗测试设备分类页的产品卡片和分类入口。
- 产品详情页中的主图、图库、PDF 下载、FAQ、询盘模块。
- 标准号、产品对象、应用场景的基础呈现方式。
- B2B 询盘路径：从产品页到询盘按钮、联系方式或表单。

不得复制现有页面文案；旧站英文内容也不应简单翻译为新中文站内容。

## 最小补充资料清单

1. 老板确认 20 个方向的内部状态：研发中 / 技术预研 / 可定制开发评估 / 暂不公开。
2. 每个方向是否已有方案、模块、样机、测试夹具、系统图或内部资料。
3. 哪些内部进展允许公开提及，哪些只内部使用。
4. 可公开图片、PDF、系统图和技术说明清单。
5. 标准号、版本、条款方向和 Page Use 确认表。
6. 第一批 5 个页面是否允许进入“研发方向页草稿”。
